Цель: реализовать алгоритм управления конвейером с использованием циклов и условных переходов.

Задание:

1.     Напишите код в OB1, который:

При активации I0.0 запускает цикл обработки 10 объектов.

Включает выход Q0.0 на 1 секунду для каждого объекта (используйте таймер TP).

Останавливает цикл при нажатии I0.1.

2.     Добавьте условие: если датчик I0.2 неактивен, пропустить объект.

Пример кода:

NETWORK 1:

    A     I      0.0

    FP    M1.0

    L     10

    T     MW10

 

Loop_Start:

    L     MW10

    JCN   Exit

    A     I      0.2

    JC    Process

    JU    Skip

 

Process:

    L     S5T#1S

    SD    T1

    A     T1

    =     Q      0.0

 

Skip:

    L     MW10

    LOOP  Loop_Start

 

Exit: NOP 0

Инструменты: Таймеры, метки, PLCSIM.
Контрольные вопросы:

1.     Как избежать бесконечного цикла?

2.     Для чего используется оператор FP?

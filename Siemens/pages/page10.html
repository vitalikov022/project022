Цель: научиться использовать счетчики и таймеры напрямую в главном цикле программы (OB1) без создания FB.

Теоретическая часть

Счетчики в STL:

CU (Count Up) — увеличивает значение на 1 по фронту сигнала.

CD (Count Down) — уменьшает значение на 1 по фронту сигнала.

R (Reset) — сбрасывает счетчик.

Таймеры в STL:

TON (Timer On-Delay) — задержка включения.

TOF (Timer Off-Delay) — задержка выключения.

TP (Pulse Timer) — генератор импульсов.

Задание

Счетчик событий:

Реализуйте в OB1 счетчик, который увеличивает значение на 1 при каждом нажатии кнопки (I0.0).

При достижении 5 срабатываний включите выход Q0.0.

Сбросите счетчик по сигналу I0.1.

Таймер задержки:

При активации входа I0.2 включите выход Q0.1 с задержкой 3 секунды (используйте TON).

Пример кода для OB1

    // Секция 1: Счетчик нажатий кнопки

    NETWORK 1:

        A     I      0.0          // Кнопка "Считать" (I0.0)

        FP    M1.0                // Обнаружение фронта

        JCN   No_Increment        // Пропустить, если фронта нет

        L     #Counter_Value      // Загрузить текущее значение

        L     1                   // Прибавить 1

        +I   

        T     #Counter_Value      // Сохранить новое значение

    No_Increment:

        L     #Counter_Value      // Проверка достижения 5

        L     5

        > =I  

        =     Q      0.0          // Включить Q0.0 при достижении

        A     I      0.1          // Кнопка "Сброс" (I0.1)

        JCN   No_Reset

        L     0                   // Сброс счетчика

        T     #Counter_Value

    No_Reset: NOP 0

    // Секция 2: Таймер задержки

    NETWORK 2:

        A     I      0.2          // Вход "Активировать задержку" (I0.2)

        L     S5T#3S              // Установка времени 3 секунды

        SD    T1                  // Запуск таймера TON (T1)

        A     T1                  // Проверить состояние таймера

        =     Q      0.1          // Включить Q0.1 после задержки

Порядок выполнения

1.     Создайте новый проект в Simatic manager, добавьте OB1.

2.     Вставьте приведенный код в OB1.

3.     Настройте аппаратные адреса:

I0.0 — кнопка "Счет".

I0.1 — кнопка "Сброс".

I0.2 — кнопка "Задержка".

Q0.0 и Q0.1 — светодиоды для индикации.

4.     Запустите симуляцию в PLCSIM:

Нажимайте I0.0 и наблюдайте за счетчиком (значение можно мониторить через таблицу переменных).

Проверьте сброс счетчика через I0.1.

Активируйте I0.2 и убедитесь, что Q0.1 включается с задержкой.

Дополнительное задание

Реализуйте обратный отсчет:

Добавьте кнопку I0.3, которая уменьшает счетчик на 1 (используйте CD).

При достижении 0 включите выход Q0.2.

Пример модификации кода:

NETWORK 3:

    A     I      0.3          // Кнопка "Уменьшить" (I0.3)

    FP    M1.1

    JCN   No_Decrement

    L     #Counter_Value

    L     1

        -I   

    T     #Counter_Value

No_Decrement:

    L     #Counter_Value

    L     0

    <=I  

    =     Q      0.2

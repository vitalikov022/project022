Лабораторная работа 1: Основы STL. Управление входами и выходами

 

Цель: изучить структуру программы в OB1, работу с логическими операциями и симулятором PLCSIM.

Задание:

1.     Создайте проект в Simatic manager, добавьте OB1.

2.     Напишите код в STL, который:

Включает выход Q0.0 при активации входа I0.0.

Выключает Q0.0, если активен вход I0.1.

3.     Добавьте инверсию: Q0.1 должен быть включен, когда I0.0 и I0.1 неактивны.

Пример кода:

NETWORK 1:

    A     I      0.0

    AN    I      0.1

    =     Q      0.0

 

NETWORK 2:

    AN    I      0.0

    AN    I      0.1

    =     Q      0.1

Инструменты: Simatic manager, PLCSIM.
